# Project Memory

## Core
Kashie: chat-based finance tracker for small business owners.
Tone: friendly, relatable, encouraging. Light humor, no slang, no jargon.
User should feel: "I understand my money AND I enjoy using this."
Backend: Lovable Cloud (Supabase). No auth yet — open RLS policies.
